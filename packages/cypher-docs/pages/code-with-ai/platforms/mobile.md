# Mobile Apps

Use Cypher CLI from your phone to keep coding sessions moving while you are away from your desk. The mobile app connects to Cloud Agents, CypherClaw, and remote sessions from your local CLI or editor extensions.

{% callout type="info" title="Android app available now" %}
Install Cypher CLI for Android from [Google Play](https://play.google.com/store/apps/details?id=com.cypher.cypherapp).
{% /callout %}

## What you can do

The mobile app lets you:

- View and manage Cypher CLI sessions, including remote CLI and extension sessions running on your local machine.
- Chat with CypherClaw from your phone.
- Spawn Cloud Agents and code directly from the app.
- Monitor and view all non-remote sessions in one place.
- Create, onboard, and manage CypherClaw instances.
