# {% $markdoc.frontmatter.title %}

{% callout type="generic" %}
Cypher CLI is an open-source AI coding assistant that works wherever you do—in your IDE, terminal, browser, or on the go. Generate code, automate reviews, debug issues, and ship faster with AI that understands your codebase.
{% /callout %}

## Where to Use Cypher

- **In your IDE** — [VS Code](/docs/code-with-ai/platforms/vscode), [JetBrains](/docs/code-with-ai/platforms/jetbrains), Cursor, Windsurf, and other VS Code forks
- [**CLI**](/docs/code-with-ai/platforms/cli) — Run Cypher from your terminal for scripting and automation
- **Web & Mobile** — Access Cypher from your browser (coming soon) or [iOS/Android apps](/docs/code-with-ai/platforms/mobile)
- [**Slack**](/docs/code-with-ai/platforms/slack) — Chat with Cypher directly in your workspace

Your sessions sync across all of these, so you can start a task on your phone and finish it in your IDE.

## What Cypher Can Do

- [**Code with AI**](/docs/code-with-ai) — Generate, refactor, and debug code through natural conversation. Use specialized modes (Code, Architect, Debug, Ask) or create your own. Get inline suggestions with Autocomplete.
- [**Collaborate**](/docs/collaborate) — Share sessions, manage team settings, and track AI adoption across your organization.
- [**Automate**](/docs/automate) — Set up AI-powered code reviews, triage agents, and auto-fixers that open new PRs based on issues.
- [**Deploy & Secure**](/docs/deploy-secure) — Build and deploy apps directly from Cypher. Run security scans and manage issues with AI assistance.

## Quick Start

1. [Install Cypher CLI](/docs/getting-started/installing) in your preferred environment
2. [Set up authentication](/docs/getting-started/setup-authentication) or use Cypher's built-in provider
3. [Run your first task](/docs/getting-started/quickstart)

{% callout type="tip" %}
**The easiest way to configure Cypher is to ask the agent.** Just tell the agent what you want — "add this MCP server", "disable OpenAI", "add my Ollama endpoint". The agent has a built-in skill for reading and updating your `cypher.jsonc` configuration. [Learn more](/docs/getting-started/settings#configuring-with-the-agent)
{% /callout %}

New to AI coding assistants? Before learning what Cypher itself does, you can learn about agentic engineering at [path.cypher.ai](https://path.cypher.ai)

Coming from Cursor or Windsurf? See our [migration guide](/docs/getting-started/migrating)
